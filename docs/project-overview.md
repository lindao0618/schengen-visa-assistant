# 客户端核心功能（详细版）

本文档仅描述**客户端（用户端）**的核心功能与使用体验，不包含管理员后台内容。

## 1. 账号体系
- 支持邮箱注册与登录。
- 登录后进入个人中心与任务页面，未登录会引导到登录/注册。

## 2. 任务中心（统一任务列表）
- 集中展示所有签证相关任务（美签/申根/材料/行程/解释信等）。
- 任务卡包含：
  - 任务名、状态、进度、创建时间、最近更新时间
  - 失败任务可查看错误截图（如 DS‑160 自动填表失败）
  - 成功任务可直接下载生成文件
- 支持筛选（全部/进行中/失败/已完成）与搜索（任务名/文件名/任务ID）。

## 3. 美国签证（US Visa）

### 3.1 照片检测
- 上传照片后进行检测：
  - 基础检测：尺寸/格式/文件大小等
  - 官方网站检测（可选）
- 输出：
  - 检测结论（中文）
  - 检查项列表（通过/不通过）
  - 处理后照片下载与预览
- 失败时会自动翻译错误并给出简短建议。

### 3.2 DS‑160 自动填表
- 上传 Excel + 照片后自动完成 DS‑160 填表流程。
- 生成 PDF 确认页，可下载。
- 支持填写进度与任务状态追踪。
- 失败时仅提供错误截图，避免冗长日志干扰。

### 3.3 DS‑160 提交
- 提交完成后生成/下载确认页。
- 支持任务进度与下载入口。

### 3.4 AIS 注册
- 自动注册并返回结果信息。
- 任务中心可查看完成/失败状态。

## 4. 材料审核 / 材料定制
- 上传材料（如酒店、机票、保险等）。
- OCR + AI 分析，输出审核结论与原因。
- 提供任务进度与结果查看。

## 5. 行程单生成
- 输入行程需求后生成行程单。
- 可下载结果文件，并在任务列表中查看进度。

## 6. 解释信生成
- 根据用户填写信息生成解释信。
- 支持中文/英文版本，提供 Word/PDF 下载。

## 7. 典型使用流程（用户端）
- 注册/登录 → 进入签证服务页面
- 上传资料或填写表单 → 任务生成并进入任务中心
- 查看进度 → 下载文件/查看结果

## 8. 用户端入口
- 签证服务入口：`/usa-visa`
- 个人中心：`/dashboard`
