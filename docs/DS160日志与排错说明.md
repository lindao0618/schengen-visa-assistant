# DS-160 日志与排错说明

## 一、日志输出位置

| 输出 | 位置 | 说明 |
|------|------|------|
| **ds160.log** | `app/api/usa-visa/usvisa/ds160-server-package/ds160.log` | 每次步骤和异常的完整日志，追加写入 |
| **任务列表** | 前端美签页面 | 运行中显示「当前步骤 + 进度」，失败时显示「失败步骤 + 失败原因」 |
| **错误截图** | `error_{时间戳}.png` | 异常时自动截取当前页面 |
| **错误 HTML** | `error_html_{时间戳}.html` | 异常时保存的页面源码 |

## 二、步骤标识（失败时用于定位）

填表流程中的主要步骤标识：

| 步骤 | 说明 |
|------|------|
| Init | 启动浏览器 |
| Goto | 访问 DS-160 网站 |
| Location | 选择地点、验证码 |
| Personal1_* | 个人信息 1（姓名、曾用名、性别等） |
| Personal2_* | 个人信息 2（国籍、身份证等） |
| Travel_* | 旅行信息（签证类型、酒店、日期等） |
| TravelCompanions_* | 同行人 |
| PrevUSTravel_* | 之前的美国旅行 |
| AddressPhone_* | 地址和电话 |
| Passport_* | 护照信息 |
| USContact_* | 美国联系人 |
| Family_* | 家庭信息 |
| WorkEducation_* | 工作/教育 |
| CountriesVisited_* | 去过的国家 |
| Security_* | 安全与背景 |
| Photo_* | 照片上传 |
| Review_* | 复核与 PDF |

## 三、错误时如何排查

1. **看任务列表错误信息**：会显示「失败步骤: xxx」和「失败原因: xxx」
2. **看 ds160.log**：打开 `ds160-server-package/ds160.log`，搜索最新一次运行的步骤和异常
3. **看错误截图**：失败任务若提供截图，可下载查看当时页面状态
4. **根据步骤对症处理**：
   - `TravelCompanions` 失败：多为 CEAC Application Error，可稍后重试或延长 postback 等待时间
   - `Personal1` 失败：多为 Excel 数据缺失或格式不对（如曾用名选 Yes 但未填姓氏/名字）
   - `CountriesVisited` 失败：多为国家名在 country_map.xlsx 中无映射
   - `Main` 失败：多为 Excel 读取或命令行参数问题

## 四、环境变量

- `DS160_TIMING=1`：启用各阶段耗时统计，写入 `ds160_timing_{时间戳}.txt`
- `HEADLESS=true`：无头模式运行浏览器
