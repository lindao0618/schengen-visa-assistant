# 签证助手系统 - 统一启动说明

## 🚀 一键启动所有服务

我们为您提供了三种启动方式，可以同时启动所有服务：

### 方式一：Python启动脚本（推荐）
```bash
python start_all_services.py
```

**特点：**
- ✅ 智能端口检测和清理
- ✅ 服务健康状态监控
- ✅ 自动重启故障服务
- ✅ 彩色日志输出
- ✅ 优雅的服务停止

### 方式二：Windows批处理文件
```bash
start_all_services.bat
```

**特点：**
- ✅ 双击即可运行
- ✅ 自动端口清理
- ✅ 彩色控制台输出
- ✅ 服务状态检查
- ✅ 可选择打开浏览器

### 方式三：PowerShell脚本
```powershell
.\start_all_services.ps1
```

**可选参数：**
- `-SkipPortCheck` - 跳过端口检查
- `-OpenBrowser` - 自动打开浏览器
- `-StartupDelay 5` - 设置服务启动间隔（秒）

## 🚨 故障排除

### 端口占用问题
如果遇到端口占用，启动脚本会自动清理：
```
⚠️  端口 3001 已被占用，正在清理...
🔪 杀死占用端口 3001 的进程: PID 12345 (node.exe)
```

### 服务启动失败
1. **检查环境依赖** - 确保Node.js、Python已安装
2. **检查网络连接** - TLS监控需要访问外部WebSocket
3. **检查文件权限** - 确保脚本有执行权限
4. **查看错误日志** - 观察控制台输出的错误信息

### 常见错误解决
- **"command not found"** - 检查PATH环境变量
- **"permission denied"** - 使用管理员权限运行
- **"port already in use"** - 手动杀死占用进程或重启电脑

## 🔄 停止服务

### 方式一：使用Ctrl+C
在启动脚本窗口按 `Ctrl+C` 优雅停止所有服务

### 方式二：任务管理器
手动结束相关进程：
- `node.exe` (前端服务)
- `python.exe` (后端服务)

### 方式三：命令行
```bash
# Windows
taskkill /f /im node.exe
taskkill /f /im python.exe

# Linux/Mac
pkill -f "npm run dev"
pkill -f "python.*main"
```

## 📞 技术支持
